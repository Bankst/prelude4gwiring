# 4WS Connections (Group 500)

## 4WS Control Unit
There are 2 CPUs in the control unit, main and sub.  
C509 connects to the main processor, C510 the sub processor.  
Each CPU can display DTCs separately via their Warning connection.  

### Connections
The 4WS Control Unit has 2 Tyco-style pin connectors, one blade-style connector and 4 ring terminal connections.
